Kenapa Memilih OdooBot?

Solusi absensi modern yang dirancang untuk efisiensi dan kemudahan akses di mana saja.

Kilat & Efisien

Absen dalam hitungan detik. Tidak perlu lagi membuka browser atau aplikasi Odoo yang berat.

Akses Terotorisasi

Keamanan adalah prioritas. Hanya nomor Telegram yang terdaftar yang dapat melakukan absensi.

Status Real-time

Dapatkan laporan status kehadiran Anda secara instan kapanpun dibutuhkan melalui command sederhana.

Cara Penggunaan

1

Konfigurasi .env

Masukkan URL, Database, Username, dan Password Odoo Anda ke dalam file konfirgurasi.

2

Daftarkan ID Telegram

Tambahkan ID Telegram Anda ke dalam daftar allowed users agar bot recognize identitas Anda.

3

Mulai Chat

Cari bot di Telegram, ketik /start, dan Anda siap melakukan absensi otomatis!


# .env Example
ODOO_URL=https://odoo.example.com
ODOO_DB=my_company
ODOO_USERNAME=user@example.com
ODOO_PASSWORD=your_password

# Bot Config
TELEGRAM_TOKEN=123456:ABC-DEF...
ALLOWED_USERS=12345678,98765432
